Información de Uso

Esta paleta de colores está diseñada para mantener la consistencia visual en todo el sistema GestionCloud.

  • Colores del Logo: Usar para elementos de marca, encabezados principales y elementos que representen la identidad corporativa
  • Colores Primarios: Usar para elementos principales como encabezados, botones principales y navegación
  • Colores Secundarios: Usar para elementos de acento, alertas y llamadas a la acción
  • Colores Neutros: Usar para texto, bordes y elementos de fondo
  • Colores de Éxito: Usar para indicar estados positivos, confirmaciones y elementos de éxito

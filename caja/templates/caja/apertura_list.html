Sistema de Gestión de Aperturas de Caja

Las aperturas de caja son el punto de inicio de cada turno de trabajo. Aquí podrás abrir cajas, registrar el monto inicial, realizar movimientos de efectivo, y al finalizar el turno, cerrar la caja con el conteo final. Este módulo es fundamental para el control financiero y la trazabilidad de todas las operaciones de efectivo.
